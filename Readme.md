# Relyft

## Overview
Relyft is a web application built using the MERN stack (MongoDB, Express.js, React, Node.js). It aims to provide a seamless experience for users to manage their tasks efficiently.

## Features
- User Authentication
- Task Management
- Real-time Updates
- Responsive Design
- Notifications
- Dark Mode

## Installation

1. **Clone the repository:**
    ```bash
    git clone https://github.com/AdharBattulwar/Relyft.git
    cd Relyft
    ```

2. **Install server dependencies:**
    ```bash
    cd server
    npm install
    ```

3. **Install client dependencies:**
    ```bash
    cd ../client
    npm install
    ```

## Usage

1. **Start the server:**
    ```bash
    cd server
    npm start
    ```

2. **Start the client:**
    ```bash
    cd ../client
    npm start
    ```

3. **Open your browser and navigate to:**
    ```
    http://localhost:3000
    ```
